# Orioles Tracker

An interactive snapshot explorer for former Baltimore Orioles players — where
they are now, the value they've produced since leaving, and a breakdown of
every Orioles trade (with an Insights tab of charts).

**Live site:** https://julianriccio.github.io/orioles-tracker/

This repo hosts only the self-contained built view (a single `index.html` with
its data embedded). Sources: MLB Stats API, Baseball-Reference, FanGraphs, Spotrac.
